## Installing

### Download Project On Your PC
- Download from github as ZIP file.
- Download by : 
    ```bash
        git clone path
    ```
### Go to the project directory
```bash
    cp .env.example .env
```

```bash
    composer install
```

```bash
    npm install
```


```bash
   npm audit fix --force    
```


```bash
     php artisn serv      
```


```bash
   npm run dev
```

```bash
    php artisan key:generate
```

```bash
    php artisan artisan migrate:fresh --seed
```

```bash
    php artisan storage:link
```
